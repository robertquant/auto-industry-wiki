# 易捷特模式

## 定义
"易捷特模式"指**中国生产+外资品牌+全球销售**的新型合作模式，以易捷特（东风×雷诺×日产合资公司）为代表，成为中国汽车技术输出欧洲的新范式。

## 模式架构

```
┌─────────────┐     ┌─────────────┐     ┌─────────────┐
│  中国生产    │ ──→ │  外资品牌    │ ──→ │  全球销售    │
│ (成本优势)   │     │ (品牌溢价)   │     │ (规避关税)   │
└─────────────┘     └─────────────┘     └─────────────┘
```

### 核心要素

| 要素 | 说明 |
|------|------|
| **中国生产** | 利用中国供应链成本优势，电池、零部件、人力成本均低于欧洲 |
| **外资品牌** | 使用外资车企旗下品牌（如Dacia），规避欧盟反补贴关税 |
| **全球销售** | 主攻欧洲市场，承接欧洲平价EV需求 |

## 模式优势

### 对外资车企
| 优势 | 说明 |
|------|------|
| **成本压缩** | 中国供应链成本优势明显 |
| **关税规避** | 品牌归属外资，规避欧盟反补贴关税 |
| **市场响应** | 快速获得平价EV产品线 |
| **风险分担** | 与中国企业共担风险 |

### 对中国车企
| 优势 | 说明 |
|------|------|
| **产能利用** | 利用闲置产能 |
| **技术变现** | 技术输出获得收益 |
| **国际经验** | 积累出海运营经验 |
| **渠道借力** | 借助外资品牌全球网络 |

### 对欧洲市场
| 优势 | 说明 |
|------|------|
| **平价EV** | €20-25k价位段获得产品 |
| **加速转型** | 加速欧洲电动化进程 |

## 模式对比

| 模式 | 代表 | 优势 | 劣势 |
|------|------|------|------|
| **易捷特模式** | Dacia Spring | 规避关税、成本最优 | 利润分成、品牌不自主 |
| **自主品牌出海** | BYD、MG | 品牌自主、利润独享 | 关税壁垒、渠道重建 |
| **海外建厂** |比亚迪土耳其工厂 | 本土化生产、关税优惠 | 投资大、周期长 |
| **技术授权** | 大众×小鹏 | 快速落地、轻资产 | 话语权有限 |

## 行业影响

### 对中国汽车产业
- 从"产品输出"升级为"技术输出"
- 绕过贸易壁垒的新路径
- 闲置产能的新出路

### 对欧洲车企
- 快速获得平价EV产品线
- 应对Stellantis式危机的新选项
- 与中国车企从竞争走向合作

### 潜在风险
- 技术外溢失控
- 合资话语权争夺
- 欧洲政策风向变化

## 判断
易捷特模式是中国汽车技术输出欧洲的"第三条道路"，介于自主品牌出海和海外建厂之间。2026年神龙汽车复制此模式，零跑×Stellantis深化合作，标志着该模式获得行业认可。对难以独立突破欧洲关税壁垒的中国车企，这是务实选择。
