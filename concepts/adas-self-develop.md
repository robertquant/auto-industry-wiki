# 智驾自研水位

## 定义
车企在智能驾驶领域自主研发的真实能力边界。"自研"是营销话术还是核心能力，需要分层评估。

## 能力分层

| 层级 | 能自研的 | 实际情况 |
|------|----------|----------|
| 感知算法（BEV+Transformer） | 极少数 | 特斯拉、华为、百度Apollo、蔚小理头部 |
| 规划控制 | 少数 | 部分头部车企 |
| 中间件/调度 | 几乎没人做 | 买Aptiv、Vector、EB方案 |
| 底层驱动/OS | 没人自研 | QNX、Linux、Autosar |

## 真正全栈自研者
- **特斯拉**：芯片+FSD OS+算法全包
- **华为**：芯片+鸿蒙OS+ADS算法

## 所谓"自研"真相

大多数车企的"自研智驾"是：
1. 买地平线/华为/Momenta的算法框架
2. 用自己的车队数据训练
3. 调参数、做场景适配
4. 宣称"自研"

这不是造假，但也不是从零到一。

## 自研门槛

| 门槛 | 说明 |
|------|------|
| 算力成本 | 训练一次L3模型需数千张A100/GH100，几千万美元 |
| 数据门槛 | 需要百万公里以上高质量标注数据 |
| 人才稀缺 | 全球BEV算法工程师就几千人，被头部垄断 |
| 迭代速度 | 特斯拉一周一版，中国车企一个月一版都跟不上 |

## 2026年格局

| 阵营 | 策略 | 份额 |
|------|------|------|
| 华为系 | 全栈方案，车企当集成商 | ~15% |
| 地平线/Momenta | 算法框架+工具链，车企做适配 | ~40%（Momenta单家长60.2%） |
| 自研头部 | 蔚小理，核心算法自研 | ~10% |
| 其他方案 | 大疆、百度、Mobileye | ~35% |
