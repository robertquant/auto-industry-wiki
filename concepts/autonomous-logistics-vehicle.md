# 无人物流车市场

L4级自动驾驶在物流配送领域的商业化落地，成为最快实现规模应用的自动驾驶场景。

## 市场规模

| 时间节点 | 数据 |
|----------|------|
| 2025年保有量 | 3.7万台 |
| 2026年Q1出货 | 近万台 |
| 增速 | 年化50%+ |

## 场景特点

### 为何物流场景率先落地

1. **低速可控**：20-40km/h，风险低
2. **固定路线**：园区、社区、快递网点
3. **运营效率**：无人化降低人力成本60%+
4. **监管宽松**：相比Robotaxi政策阻力小

### 典型应用场景

| 场景 | 代表企业 |
|------|----------|
| 快递配送 | [[dongfeng-openvan]]、[[jiushi-intelligence]] |
| 园区物流 | 京东、顺丰 |
| 商超配送 | 盒马、美团 |

## 商业模式

| 模式 | 说明 | 代表 |
|------|------|------|
| 车辆销售 | 卖车一次性收入 | 东风OpenVAN |
| Inside | 技术授权/系统销售 | 九识智能 |
| 运营服务 | 按单/按里程收费 | 京东物流 |

## 与Robotaxi对比

| 维度 | 无人物流车 | Robotaxi |
|------|-----------|----------|
| 时速 | 20-40km/h | 80-120km/h |
| 场景 | 封闭/半封闭 | 开放道路 |
| 监管 | 宽松 | 严格 |
| 商业化 | 已盈利 | 仍亏损 |
| 代表 | 九识智能/东风 | [[pony-ai]]/[[weRide]] |

## 发展趋势

1. **成本下降**：激光雷达价格破2000元，整车成本5年内降50%
2. **规模化加速**：2026年行业出货量预计超3万台
3. **场景扩展**：从快递到商超、园区、工厂物流
4. **政策利好**：多城市开放路权试点
