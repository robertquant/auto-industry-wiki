# 人形机器人

## 定义
具备人类形态的智能机器人，双足行走、双臂操作、头部感知。车企布局人形机器人作为第二增长曲线，利用造车积累的供应链和技术能力。

## 车企布局动机

| 维度 | 说明 |
|------|------|
| 技术复用 | 智驾的感知、决策、控制能力直接迁移 |
| 供应链复用 | 电机、电池、执行器与汽车共用 |
| 制造能力 | 车企的精密制造和量产能力 |
| 增长天花板 | 汽车市场饱和，机器人开辟新空间 |

## 技术复用分析

| 汽车技术 | 机器人应用 |
|----------|------------|
| BEV感知 | 环境理解 |
| 端到端决策 | 运动规划 |
| 电机控制 | 关节驱动 |
| 电池系统 | 动力源 |
| OTA升级 | 功能迭代 |

## 商业化时间线

| 阶段 | 时间 | 场景 |
|------|------|------|
| 技术验证 | 2022-2024 | 实验室演示 |
| 工厂部署 | 2024-2026 | 车厂内部使用 |
| 商业落地 | 2026-2028 | 物流、服务业、工业（奇瑞墨茵M1已量产） |
| 家庭普及 | 2028+ | 高收入家庭 |

## 产业判断

### 乐观因素
- 技术复用降低研发成本
- 车企制造能力支撑量产
- AI能力提升加速智能化
- 奇瑞墨茵M1已实现28.58万元量产价格

### 挑战因素
- 成本仍高（特斯拉Optimus尚未公布量产价格）
- 续航和可靠性待突破
- 应用场景需要验证

## 最新进展（2026年6月）

### 智元机器人远征A3
- **里程碑**：全球首个自主打乒乓球的全尺寸双足人形机器人
- **技术突破**：实时感知-决策-执行闭环，毫秒级响应
- **意义**：人形机器人从"静态展示"进入"动态交互"阶段

参见：[[agxbot]]（如有）

## 判断
人形机器人是车企的第二增长曲线，奇瑞墨茵M1的量产突破标志着商业化进入实质阶段。28.58万元的售价已进入工业场景可接受区间。小鹏IRON定于2026年底量产，车企"造人"竞赛正式开启。智元远征A3的乒乓球能力展示标志着技术从"演示"进入"实用"阶段。2026年的正确姿势是：量产验证+工业落地，消费级市场仍需时间。
