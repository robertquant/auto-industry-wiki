# 极氪领克整合（2026年6月）

## 事件概述

2026年6月，吉利宣布极氪与领克合并为"高端豪华新能源汽车集团"，冲击2026年百万产销目标。这是《台州宣言》战略整合的核心落地动作。

## 整合前数据（Q1）

| 品牌 | Q1销量 | 同比 | 成交均价 | 定位 |
|------|--------|------|----------|------|
| 极氪 | 7.7万辆 | +86% | 近30万 | 高端纯电 |
| 领克 | 8.2万辆 | +12% | - | 燃油+混动转型 |
| 银河 | 23.9万辆 | - | - | 新能源主力 |

## 整合背景

### 《台州宣言》五大战略支柱

| 战略 | 内核 |
|------|------|
| 聚焦 | 品牌矩阵精简，资源集中 |
| 整合 | 供应链、研发、智驾一体化 |
| 协同 | 各品牌平台共享 |
| 稳健 | 盈利优先于规模扩张 |
| 人才 | 核心团队稳定与激励 |

### 整合前问题

- **5个智驾团队**：极氪、领克、银河、路特斯、其他各做各的
- **技术路线分散**：重复研发、资源浪费
- **品牌内耗**：价格带重叠、用户分流

## 整合后战略

### 智驾团队整合

- **整合前**：5个山头，各自为战
- **整合后**：重庆千里智驾统一平台
- **团队规模**：近3000人
- **预期效果**：研发效率提升30%+

### 品牌定位

- **极氪**：纯电豪华，均价35万+
- **领克**：燃油+混动转型，向新能源过渡
- **银河**：新能源主力，10-20万级市场

### 2026年目标

- **产销目标**：100万辆（整合后）
- **海外目标**：75万辆出口
- **高端化指标**：极氪9系/8系占比近50%

## 行业意义

1. **从扩张到集约**：吉利的战略转变，从多品牌扩张转向集约化运营
2. **避免内耗**：统一研发、供应链、智驾，避免品牌间重复投入
3. **高端化突破**：极氪均价近30万、旗舰占比近50%，高端化确立
4. **对标BBA**：整合后的高端集团直接竞争奔驰、宝马、奥迪

## 市场表现（2026年5月）

- **极氪5月交付34,377辆，同比+81.8%**
- **旗舰车型占比近50%**：9系/8系高端化成功
- **海外进入法国市场**：葡萄牙、奥地利签约
