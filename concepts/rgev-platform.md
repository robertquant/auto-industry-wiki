# RGEV平台 (Renault Global Electric Vehicle)

## 概述
雷诺下一代纯电平台，代号RGEV Medium 2.0，目标追平中国供应链效率。2026年5月披露技术规格。

## 技术规格

| 参数 | 规格 |
|------|------|
| 平台电压 | 800V高压架构 |
| 纯电续航 | 750km（WLTP） |
| 增程续航 | 1400km（含增程器） |
| 目标 | 追平中国供应链效率 |

## 战略定位

### futuREady计划核心
- 雷诺2030年电动化战略的技术底座
- 承载欧洲100%电动化目标
- 对标中国BEA/SEA/Neue Klasse等新一代平台

### 降本目标
- **核心挑战**：欧洲电动车成本高于中国30-40%
- **技术路线**：800V架构+高集成度电驱+规模化采购
- **本地化生产**：欧洲本土工厂降低关税和物流成本

## 时间线

| 时间 | 里程碑 |
|------|--------|
| 2026.5 | 技术规格首次披露 |
| 2028 | 计划启用（futuREady规划） |
| 2030 | 全面量产 |

## 竞争格局

### 对标平台
- 大众MEB+（升级版）
- 宝马Neue Klasse
- 奔驰MB.EA Medium
- 中国：比亚迪BEA、吉利SEA、小米摩德纳

### 核心差距
- 中国平台迭代周期12-18个月 vs 欧洲36-48个月
- 中国800V已量产渗透，欧洲2026年才规模化
- 中国电池成本优势明显（宁德时代、比亚迪垂直整合）

## 技术意义

RGEV平台是雷诺"去中国供应链依赖"与"不得不依赖中国效率"矛盾的具体体现。技术规格追平中国，但量产时间落后2-3年，关键问题在于欧洲能否建立与中国等效的供应链生态。
