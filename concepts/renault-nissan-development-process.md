# 雷诺-日产联盟整车开发流程

> 雷诺-日产联盟特有的整车开发里程碑体系，与通用/上汽G节点体系、大众PEPE流程不同。

## 核心里程碑

| 缩写 | 英文全称 | 中文含义 | 阶段说明 |
|------|----------|----------|----------|
| **CF** | Concept Freeze | 概念冻结 | 造型方向确定，设计参数锁定 |
| **VCO** | Vehicle Confirmation Order | 车辆确认订单 | 项目正式立项批准，商业决策点 |
| **GO** | Go Decision | 生产启动批准 | 量产前的最终决策点 |
| **ABPT** | Advanced Build Pilot Test | 试生产验证 | 工装样车试制阶段 |
| **PPC** | Production Pilot Car | 试生产车 | 预试生产阶段 |
| **SOP** | Start of Production | 量产启动 | 正式投产 |

## 流程时间线

```
CF → VCO → GO → ABPT → PPC → SOP
↑        ↑       ↑
概念冻结  立项    生产批准
```

## 样车类型

### BoxCar（盒车）

| 特征 | 说明 |
|------|------|
| 定义 | 早期总布置验证样车 |
| 外形 | 方块状/盒状，无造型设计 |
| 阶段 | CF（概念冻结）之前或同期 |
| 目的 | 验证动力总成布置、底盘架构、人机工程 |
| 零件状态 | 手工/快速成型件 |

### 样车演进序列

| 样车类型 | 零件状态 | 验证目的 | 外形状态 |
|----------|----------|----------|----------|
| **BoxCar** | 手工/快速成型 | 总布置、架构验证 | 方块状，无造型 |
| **Mule Car** | 简易工装/手工 | 底盘调校、动力匹配 | 改装现有车型 |
| **Simulator Car** | 软模/手工 | 系统性能模拟 | 接近量产造型 |
| **EP** | 工程样件 | 性能、可靠性验证 | 量产造型 |
| **PPV** | 正式工装 | 产品+工艺验证 | 量产状态 |

## 与其他车企流程对比

| 雷诺-日产 | 通用/上汽 | 大众 | 含义 |
|-----------|-----------|------|------|
| CF | G7（方案批准） | A阶段结束 | 概念冻结 |
| VCO | G6（项目批准） | B阶段启动 | 立项批准 |
| GO | G4/G3 | D阶段 | 生产批准 |
| ABPT | PPV | VS阶段 | 试生产验证 |
| PPC | PP | 0系列 | 预试生产 |
| SOP | SOP | SOP | 量产启动 |
| BoxCar | Mule Car同期 | 包样车 | 布置验证 |

## 关键特点

1. **VCO是核心商业决策点**：一旦VCO批准，项目进入实质性开发阶段
2. **GO是量产准入**：通过GO评审后才能进入试生产阶段
3. **BoxCar是联盟特有术语**：用于早期架构验证，降低后期变更风险
4. **ABPT与PPC区分**：ABPT是工装样车试制，PPC是预试生产，逐步逼近量产状态
