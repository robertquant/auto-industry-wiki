# Web端可视化建模工具

浏览器端可视化编程/建模工具，降低开发门槛，支持实时仿真。

## 主流工具对比

| 工具 | 特点 | 生成语言 | 部署方式 |
|------|------|----------|----------|
| **Pictorus** | 浏览器端建模、OTA部署、实时遥测 | Rust/C/C++ | 嵌入式设备 |
| **PathSim** | 开源、Pyodide/WebAssembly | Python/C | 本地/云端 |
| **Node-RED** | IoT流程编程 | JSON/Node.js | 本地服务器 |
| **Rete.js** | 可定制框架 | 自定义 | Web应用 |

## Pictorus深度分析

### 核心优势

- **浏览器端建模**：无需安装IDE
- **OTA部署**：模型更新直接推送到设备
- **实时遥测**：运行数据实时回传
- **原型开发免费**：商业化前无成本

### 技术架构

```
浏览器建模 → Rust代码生成 → 编译 → OTA部署 → 嵌入式设备运行
```

### 应用场景

- 快速原型开发
- IoT设备编程
- 实验性项目

## 与传统工具对比

| 维度 | 传统（MATLAB/Simulink） | Web工具 |
|------|------------------------|---------|
| 安装成本 | 高（许可证+安装） | 低（浏览器打开） |
| 学习曲线 | 陡峭 | 平缓 |
| 实时部署 | 复杂 | 简单（OTA） |
| 生态成熟度 | 高 | 发展中 |
| 行业认可 | ISO 26262认证 | 有待验证 |

## 行业意义

- **降低门槛**：非专业开发者可参与建模
- **加速迭代**：浏览器端开发+OTA部署
- **挑战传统**：Simulink许可证模式受冲击
- **安全质疑**：ASIL-D等级是否适用有待验证
