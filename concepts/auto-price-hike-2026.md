# 2026车企涨价潮

## 概述
2026年4-5月，新能源汽车行业从价格战转向涨价潮，标志着行业利润修复周期的开始。

## 触发因素

### 存储芯片价格暴涨
| 存储类型 | 涨幅 |
|----------|------|
| 高端DDR5现货 | **+300%** |
| 通用型DRAM | +55-60% |
| 车规级存储 | 被AI算力产业挤占产能 |

**传导链条**：AI大模型训练消耗HBM → 存储芯片产能被抽离 → 车载存储需求上升 → 价格暴涨

### 行业利润率触红线
| 指标 | 数据 |
|------|------|
| 2026年Q1汽车行业利润率 | **3.2%** |
| 同比变化 | 从4%降至3.2% |
| 行业利润同比 | **-18%** |
| 对比工业平均净利率 | 6%（车企仅为半） |

| 趋势 | 说明 |

| **智能化的代价** | AI训练推高芯片价格→传导到车价 |
| **消费者买单** | 智能化成本最终转移给终端用户 |

## 判断
2026年4-5月是行业转折点，价格战结束、涨价潮全面开启。

### 深层原因
1. **行业利润率仅2.9%**：不可持续
2. **购置税减半退坡**：政策红利结束
3. **监管禁止恶意低价倾销**：行政干预
4. **存储芯片暴涨**：成本传导

从"价格战→涨价潮"的转变标志着新能源汽车行业进入利润修复周期。比亚迪智驾包涨价只是开始，整车涨价已成常态。
