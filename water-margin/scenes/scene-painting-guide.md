# 水浒传场景绘制指南

## 一、鲁智深拳打镇关西

---

### 场景基本信息

| 项目 | 内容 |
|------|------|
| 地点 | 渭州郑屠肉铺 |
| 时间 | 白天，午时 |
| 天气 | 晴朗 |
| 人物 | 鲁智深、郑屠、金翠莲父女（背景）、围观者 |

---

### 场景环境

#### 肉铺布局
```
┌─────────────────────────────────────┐
│              街道                    │
├─────────────────────────────────────┤
│  [招牌] 郑屠肉铺                     │
│┌─────────────┬─────────────┐       │
││   肉案       │   内室       │       │
││  [刀][肉]   │[桌][椅]│       │
││   [案板]    │             │       │
│└─────────────┴─────────────┘       │
│                门                   │
└─────────────────────────────────────┘
```

#### 环境细节
| 元素 | 描述 |
|------|------|
| 招牌 | "郑屠肉铺"木牌，悬挂门上 |
| 肉案 | 木质案板，上面摆猪肉、刀 |
| 刀具 | 屠刀、剁肉刀，挂在墙上或案上 |
| 肉块 | 猪肉、排骨，挂在铁钩上 |
| 地面 | 青石板，有血迹 |
| 围观者 | 街坊四邻，围观 |

---

### 人物动态

#### 鲁智深
| 项目 | 描述 |
|------|------|
| 位置 | 肉案前，踏住郑屠 |
| 姿势 | 右脚踏住郑屠胸口，右手举拳 |
| 表情 | 怒目圆睁，咬牙切齿 |
| 动作 | 挥拳猛打 |
| 服饰 | 僧袍、僧帽、僧鞋 |

#### 郑屠
| 项目 | 描述 |
|------|------|
| 位置 | 肉案前，倒在地上 |
| 姿势 | 仰面朝天，双手护住头部 |
| 表情 | 惊恐、痛苦 |
| 动作 | 挣扎求饶 |
| 服饰 | 短褐、围裙、布鞋 |

#### 三拳详解

| 拳数 | 部位 | 效果 | 画面表现 |
|------|------|------|---------|
| 第一拳 | 鼻子 | 鼻梁打折，鲜血迸流 | 鲜血从鼻孔喷出 |
| 第二拳 | 眼睛 | 眼棱缝裂，乌珠迸出 | 眼角流血 |
| 第三拳 | 太阳穴 | 脑浆迸流，当场毙命 | 头部变形，不再动弹 |

---

#### 主体构图

```
│    [鲁智深]                          │

│       │                              │

│       │  ↓                           │
│    [郑屠] ←── 踏住                   │

```

#### 视角选择
| 视角 | 效果 |
|------|------|
| 正面视角 | 展现鲁智深愤怒表情 |
| 侧面视角 | 展现打拳动态 |
| 俯视视角 | 展现场景全貌 |
| 仰视视角 | 突出鲁智深威武 |

---

### 光影氛围

| 元素 | 处理方式 |
|------|---------|
| 光源 | 自然光，从门口射入 |
| 明暗 | 鲁智深受光，郑屠背光 |
| 氛围 | 紧张、激烈 |
| 色调 | 暖色调为主，血迹红色 |

| 6 | 挥拳猛打 | 右手拳头猛打 |
