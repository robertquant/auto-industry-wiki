### 青龙偃月刀（关胜）

#### 基本信息
| 项目 | 数据 |
|------|------|
| 全长 | 约2米 |
| 刀身长 | 约70厘米 |
| 刀背厚 | 约1.5厘米 |
| 重量 | 82斤（演义），实际约20-30斤 |
| 材质 | 铁制 |

#### 造型特征
| 部位 | 描述 |
|------|------|
| 刀身 | 月牙形，刀背有青龙图案 |
| 刀刃 | 内凹外凸，如偃月 |
| 刀柄| 长柄，末端有刀鐏 |
| 配饰 | 刀身有龙纹，刀柄缠丝 |

#### 绘画要点
- 刀身呈月牙形，外凸为刃
- 刀背有青龙浮雕纹饰
- 刀柄长，需要双手握持
- 整体呈绿色调（青龙意象）
- 刀刃银白，刀背青绿

## 五、兵器绘制技法

### 金属质感

| 材质 | 颜色 | 光泽 | 反光 |
|------|------|------|------|
| 铁制兵器 | 铁灰、黑色 | 金属光泽 | 强反光 |
| 钢制兵器 | 银白、灰白 | 金属光泽 | 强反光 |
| 铜制兵器 | 金黄、红铜 | 金属光泽 | 中等反光 |
| 木质兵器 | 褐色、棕色 | 无光泽 | 无反光 |

### 刃口表现

| 部位 | 表现方式 |
|------|---------|
| 刀刃 | 细白线，有反光 |
| 刀背 | 粗线条，有厚度 |
| 血槽 | 凹陷，有阴影 |
| 刀尖 | 锐利三角形 |

### 兵器透视

| 角度 | 表现要点 |
|------|---------|
| 正面 | 兵器完整形态 |
| 侧面 | 兵器厚度 |
| 斜角 | 兵器立体感 |
| 远近 | 大小变化 |

## 六、兵器使用动态

### 长兵器动态

| 动作 | 描述 |
|------|------|
| 刺 | 兵器向前刺出，手臂伸直 |
| 劈| 兵器从上向下劈砍 |
| 挑 | 兵器从下向上挑起 |
| 架 | 兵器横架，格挡攻击 |
| 扫 | 兵器横扫，攻击侧面 |

### 短兵器动态

| 动作 | 描述 |
|------|------|
| 砍 | 兵器从上向下砍 |
| 劈 | 兵器斜向劈砍 |
| 挡 | 兵器横挡，格挡攻击 |
| 刺 | 兵器向前刺出 |
| 扫| 兵器横扫，攻击侧面 |

### 远射兵器动态

| 动作 | 描述 |
|------|------|
| 拉弓 | 一手持弓，一手拉弦 |
| 瞄准 | 眼睛、箭头、目标三点一线 |
| 射箭 | 松弦，箭飞出 |
| 持弩 | 一手持弩，一手扣扳机 |
