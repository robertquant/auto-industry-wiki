# 宋朝古玩与文房四宝

## 一、宋朝文房四宝

### 1. 端砚（端溪砚）

#### 产地
- **地点**：广东肇庆（古称端州）
- **开采**：端溪水岩

#### 特点
| 特征 | 描述 |
|------|------|
| 石质 | 细腻温润 |
| 颜色 | 紫色为主，有青、绿、白等 |
| 石品 | 鱼脑冻、蕉叶白、天青、猪肝紫 |
| 发墨 | 发墨快、不损毫 |

#### 名品
- **鱼脑冻**：石中有白色冻状物，如鱼脑
- **蕉叶白**：白色条纹如蕉叶
- **天青**：青色石品
- **火捺**：红色斑点

#### 宋代端砚特征
- 造型古朴大方
- 崇尚自然
- 不事雕琢
- 讲究石品

### 2. 歙砚（龙尾砚）

#### 产地
- **地点**：安徽歙县（古称歙州）
- **开采**：龙尾山

#### 特点
| 特征 | 描述 |
|------|------|
| 石质 | 坚润 |
| 颜色 | 黑色为主|
| 石品 | 眉子、金星、银星 |
| 发墨 | 发墨细腻 |

#### 名品
- **金星**：石中有金色斑点
- **银星**：石中有银色斑点
- **眉子**：条纹如眉毛
- **罗纹**：纹如罗绮

### 3. 洮砚

#### 产地
- **地点**：甘肃洮州
- **开采**：洮河河床

#### 特点
| 特征 | 描述 |
|------|------|
| 石质 | 细膩温润 |
| 颜色 | 绿色为主 |
| 石品 | 绿漪石、黄膘 |
| 发墨 | 发墨快 |

### 4. 澄泥砚

#### 产地
- **地点**：山西绛州
- **工艺**：陶土烧制

#### 特点
| 特征 | 描述 |
|------|------|
| 工艺 | 淘洗、沉淀、制坯、烧制 |
| 颜色 | 朱、紫、黄、绿 |
| 质地 | 坚密如石 |
| 发墨 | 发墨细腻 |

## 二、宋朝名墨

### 1.徽墨

#### 产地
- **地点**：安徽歙县（古称徽州）
- **特点**：色泽黑润，光泽如漆

#### 制作工艺
- **原料**：松烟、桐油烟、漆烟等
- **配料**：麝香、冰片、金箔等
- **工艺**：炼烟、和胶、杵捣、成型、晾晒

#### 名品
- **李廷圭墨**：南唐制墨名家，宋人珍视
- **潘谷墨**：宋代制墨名家

### 2. 墨的等级

| 等级 | 特点 | 用途 |
|------|------|------|
| 御墨 | 皇家专用 | 皇帝御用 |
| 贡墨 | 进贡朝廷 | 官府使用 |
| 文人墨 | 讲究品质 | 士人使用 |
| 民用墨 | 普通品质 | 平民使用 |

## 三、宋朝名纸

### 1. 宣纸

#### 产地
- **地点**：安徽泾县（古称宣州）

#### 特点
| 特征 | 描述 |
|------|------|
| 原料 | 青檀皮、沙田稻草 |
| 质地 | 绵软坚韧 |
| 吸墨 | 吸墨均匀|
| 耐久 | 寿命千年 |

#### 种类
- **生宣**：未经矾水处理，吸墨强
- **熟宣**：经矾水处理，不吸墨
- **半熟宣**：介于两者之间

### 2. 澄心堂纸

#### 产地
- **地点**：南唐澄心堂
- **特点**：肤如卵膜，坚洁如玉

#### 历史
- 南唐后主李煜创制
- 宋人极为珍视
- 产量稀少，价值连城

### 3. 薛涛笺

#### 产地
- **地点**：四川成都

#### 特点
| 特征 | 描述 |
|------|------|
| 颜色 | 红、粉、绿等 |
| 用途 | 写诗、信札 |
| 特点 | 小巧精致 |

## 五、宋朝古玩

### 1. 瓷器

#### 五大名窑

| 窑口 | 地点 | 特点 |
|------|------|------|
| 汝窑 | 河南汝州 | 天青色、蟹爪纹 |
| 官窑 | 河南开封 | 紫口铁足、冰裂纹 |
| 哥窑 | 浙江龙泉 | 金丝铁线 |
| 钧窑 | 河南禹州 | 窑变釉、蚯蚓走泥纹 |
| 定窑 | 河北曲阳 | 白瓷、刻花 |

#### 汝窑
- **颜色**：天青色（雨过天青云破处）
- **釉面**：蟹爪纹、蝉翼纹
- **数量**：极少，宋人已珍视
- **价值**："家有万贯，不如汝瓷一片"

#### 官窑
- **颜色**：青、灰、米黄
- **釉面**：冰裂纹、紫口铁足
- **用途**：皇家专用

#### 哥窑
- **颜色**：青、灰、米黄
- **釉面**：金丝铁线（开片纹）
- **特点**：纹片如冰裂

#### 钧窑
- **颜色**：玫瑰紫、海棠红、月白
- **釉面**：窑变釉、蚯蚓走泥纹
- **特点**："入窑一色，出窑万彩"

#### 定窑
- **颜色**：白瓷为主
- **装饰**：刻花、印花
- **特点**：釉面温润

### 2. 青铜器

#### 类型
| 类型 | 用途 | 特点 |
|------|------|------|
| 鼎 | 祭祀 | 三足或四足 |
| 尊 | 盛酒 | 造型多样 |
| 爵 | 饮酒 | 三足、流、尾 |
| 觯 | 饮酒 | 圆形或方形 |
| 盉| 盛酒 | 有流、有鋬 |

#### 宋代仿古青铜器
- **特点**：仿商周青铜器
- **用途**：祭祀、陈设
- **工艺**：精良

### 3. 玉器

#### 类型
| 类型 | 用途 | 特点 |
|------|------|------|
| 玉璧 | 礼器 | 圆形，中间有孔 |
| 玉琮 | 礼器 | 外方内圆 |
| 玉璜 | 礼器 | 半圆形 |
| 玉佩 | 佩饰 | 形状多样 |
| 玉带钩 | 实用 | 革带配件 |

#### 宋代玉器特点
- 造型写实
- 纹饰精细
- 题材世俗化
