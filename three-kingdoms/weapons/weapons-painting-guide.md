### 青龙偃月刀（关羽）

#### 基本信息
| 项目 | 数据 |
|------|------|
| 全长 | 约2米 |
| 刀身长 | 约70厘米 |
| 刀刃宽 | 约20厘米 |
| 刀背厚 | 约1.5厘米 |
| 重量 | 82斤（演义），实际约10-15公斤 |
| 材质 | 铁制 |

#### 造型特征
| 部位 | 描述 |
|------|------|
| 刀身 | 月牙形，刀背有青龙图案 |
| 刀刃 | 内凹外凸，如偃月 |
| 刀背 | 厚实，有青龙浮雕 |
| 刀柄 | 长柄，末端有刀鐏 |
| 刀鐏 | 金属套，可插入地面 |

#### 详细尺寸
```
┌─────────────────────────────────────┐
│              刀身                    │
│         ┌─────────────┐             │
│        ╱               ╲            │

│              │←约130cm→             │
│              │                      │
│              ▼刀鐏                  │
└─────────────────────────────────────┘
```

#### 绘画要点
- 刀身呈月牙形，外凸为刃
- 刀背有青龙浮雕纹饰（重要特征）
- 刀柄长，需要双手握持
- 整体呈绿色调（青龙意象）
- 刀刃银白，刀背青绿
- 刀鐏金属光泽

#### 金属质感
| 部位 | 颜色 | 光泽 | 特点 |
|------|------|------|------|
| 刀刃 | 银白 | 强反光 | 锋利 |

| 刀柄 | 深红 | 无光泽 | 缠丝 |
| 刀鐏 | 金色 | 强反光 | 金属套 |

#### 使用动态
| 动作 | 描述 |
|------|------|
| 挥砍 | 双手握柄，从上向下劈砍 |
| 横扫 | 双手握柄，横向扫击 |
| 挑击 | 刀刃向上挑起 |
| 格挡 | 刀背格挡敌方攻击 |

## 四、兵器绘制技法

### 金属质感表现

| 材质 | 颜色 | 光泽 | 反光 |
|------|------|------|------|
| 铁制兵器 | 铁灰、黑色 | 金属光泽 | 强反光 |
| 钢制兵器 | 银白、灰白 | 金属光泽 | 强反光 |
| 铜制兵器 | 金黄、红铜 | 金属光泽 | 中等反光 |
| 木质兵器 | 褐色、棕色 | 无光泽 | 无反光 |
| 缠丝 | 深红、黑色 | 无光泽 | 无反光 |

### 刃口表现

| 部位 | 表现方式 |
|------|---------|
| 刀刃 | 细白线，有反光 |
| 刀背 | 粗线条，有厚度 |
| 血槽 | 凹陷，有阴影 |
| 刀尖 | 锐利三角形 |

### 兵器透视

| 角度 | 表现要点 |
|------|---------|
| 正面 | 兵器完整形态 |
| 侧面 | 兵器厚度 |
| 斜角 | 兵器立体感 |
| 远近 | 大小变化 |

### 动态表现

| 动作 | 描述 |
|------|------|
| 挥砍 | 兵器从上向下劈砍 |
| 刺击 | 兵器向前刺出 |
| 格挡 | 兵器横挡 |
| 横扫 | 兵器横向扫击 |
