# 官渡之战连环画分镜指南

## 概述

官渡之战（200年）是曹操统一北方的决定性战役，以少胜多的经典战例。本指南将战役分为15个核心场景，共计**45-60张连环画分镜**。

---

## 战役基础信息

| 项目 | 内容 |
|------|------|
| 时间 | 200年 |
| 地点 | 官渡（今河南中牟东北） |
| 对阵 | 曹操 vs 袁绍 |
| 兵力 | 曹操约2-3万 vs 袁绍约10万+ |
| 结果 | 曹操以少胜多，统一北方 |

## 场景一：青梅煮酒论英雄（4张）

**场景概述**：曹操试探刘备，刘备韬光养晦。

### 分镜1：梅园对饮

| 项目 | 描述 |
|------|------|
| 地点 | 许都丞相府后园 |
| 时间 | 初夏午后 |
| 人物 | 曹操、刘备 |
| 构图 | 中景，二人对坐 |
| 动作 | 曹操举杯，刘备谨慎陪饮 |
| 表情 | 曹操豪迈试探，刘备小心翼翼 |
| 氛围 | 表面轻松，暗流涌动 |

**绘制要点**：梅树上青梅累累，酒案上青梅、酒壶。曹操便服但气场威严，刘备素色便服低调。午后阳光，梅树投下斑驳光影。
