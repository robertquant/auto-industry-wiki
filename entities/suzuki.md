# 铃木 (Suzuki)

## 概述
日本小型车专家，印度市场绝对统治者。Maruti Suzuki占据印度~40%市场份额，2026年宣布72亿美元加码印度产能布局。

## 印度市场地位

| 指标 | 数据 |
|------|------|
| 市场份额 | ~40%（通过Maruti Suzuki合资） |
| 投资额（2026） | 72亿美元（最大单笔投资） |
| 定位 | 印度国民车，价格敏感市场霸主 |

### 成功要素

| 因素 | 说明 |
|------|------|
| **先发优势** | 1980年代进入，与印度政府建立深厚关系 |
| **产品定位** | 小型车、高性价比，契合印度市场需求 |
| **本地化深度** | 产能、供应链、研发全面本地化 |
| **政治友好** | 日系身份，无地缘政治摩擦 |

## 2026年战略升级

### 72亿美元投资方向

- 扩大产能：满足印度汽车消费增长
- 电动化布局：混动/纯电双线
- 供应链强化：本土化率进一步提升

### 与丰田协同

- 丰田18亿美元投资印度，与铃木共享基础设施
- 两家日系车企形成印度市场联盟
- 互补定位：铃木小型车 + 丰田混动技术

## 地缘政治护城河

> **日系在印度的真正优势不是技术，是政治**

2020年加勒万河谷冲突后，印度对中国投资审查趋严：
- 中国车企（长城、比亚迪）被政策性锁死
- MG被迫交出51%控股权
- 日系享有天然政治优势，投资审查友好

铃木72亿美元投资本质是押注"中国EV进不来"的政治确定性。
