# 蔚来神玑NX9031芯片

蔚来汽车自研智驾芯片，搭载于乐道L60中高配车型。

## 产品规格

| 参数 | 规格 |
|------|------|
| 制程工艺 | **5nm车规级** |
| 单芯算力 | **1000+ TOPS** |
| 定位 | 车企自研智驾芯片旗舰 |

## 技术架构

### 蔚来世界模型NWM
- 基于神玑NX9031芯片搭载
- 支持场景预测与决策
- 从"感知-规划-控制"走向场景预测与生成

### Coconut椰子原生智能体架构
- 蔚来自研智能体架构
- 支持多模态交互与决策

### SkyOS·天枢AI操作系统
- 蔚来自研智驾操作系统
- 全域领航辅助NOP+，支持自主进出换电站

## 产品定位

神玑NX9031是蔚来从Tier-1供应商转向核心技术自研的战略产品。蔚来作为新势力三强之一，与[[byd-xuanji-a3]]、[[li-auto-m100]]、小鹏图灵芯片共同构成2026年车企自研芯片量产浪潮。

## 搭载车型

- 乐道L60中高配
- 后续车型待公布

## 行业意义

### 自研芯片破局信号
1. **自研芯片规模化降本**：从蔚来高端专属下放至乐道品牌
2. **挑战英伟达主导地位**：5nm + 1000+ TOPS参数挑战Orin-X
3. **算力利用率比峰值算力更重要**：蔚来方案优化车端算力利用率
