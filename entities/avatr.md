# 阿维塔 (Avatr)

## 概述
阿维塔是长安汽车、华为、宁德时代三方合作的智能电动汽车品牌，定位高端市场。2026年推出07L车型，搭载华为乾崑智驾系统。

## 股权结构

| 股东 | 角色 |
|------|------|
| 长安汽车 | 整车制造、产品定义 |
| 华为 | 智能驾驶、智能座舱 |
| 宁德时代 | 动力电池 |

## 07L车型（2026年）

### 智驾配置
- 搭载华为乾崑智驾系统
- **896线激光雷达**（当前量产最高规格）
- 全量智能化系统（智能驾驶+智能座舱+智能车控）

### 定位
- 高端智能电动SUV
- 与问界M9同属华为智选车生态

## 华为合作深度

阿维塔是华为HI模式的标杆合作方：
- 华为提供智驾全栈解决方案
- 与问界、智界、尚界形成华为智选车矩阵
- 区别于智选车模式，阿维塔保留更多品牌自主性

## 行业意义
