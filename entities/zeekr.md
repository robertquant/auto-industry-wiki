# 极氪 (Zeekr)

## 概述
吉利集团旗下高端新能源品牌，定位30万+豪华市场。2021年成立，目前是中国品牌高端化最成功的案例之一。

## 核心数据（2026年5月核实）

### 品牌整体表现
| 指标 | 数据 |
|------|------|
| 2026年4月交付 | **31,787辆**，同比+132%，创历史新高 |
| 连续增长 | 连续3个月同环比双增长（新势力唯一） |
| 累计交付 | 突破75万辆（截至2026年4月） |
| 单车均价 | 接近35万元 |

### 50万以上市场表现
- **市占率约30%**：中国50万以上新能源市场，每10辆有3辆是极氪
- 连续5个月蝉联50万以上大型SUV销冠

## 战略定位

### 高端化成功的原因
1. **产品力硬核**：浩瀚架构、三电技术、性能标杆
2. **价格带精准**：30-50万区间，避开BBA入门但有溢价空间
3. **用户转化**：80%+来自传统豪华品牌，说明品牌认可度高
4. **交付提速**：供应链问题解决后，Hyper版从等数月缩短到5-8周

### 与银河的关系
- **银河**：10-15万级，走量性价比路线
- **极氪**：30-50万+级，高端豪华路线
- 两者**不是内卷关系**，而是覆盖不同价格带的协同布局

## 重要提醒

⚠️ **数据核实要点**：
- 2026年4月极氪交付是**+132%**，不是"-14%"
- 高端化非常成功，不存在"高端化遇阻"
- 单车均价35万、9X均价53万，是目前中国品牌中高端化最成功的之一
