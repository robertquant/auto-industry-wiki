# 理想马赫M100芯片

理想汽车自研智驾芯片，单芯算力1280 TOPS，5nm工艺，利用率达82%。

## 核心参数

| 参数 | 规格 |
|------|------|
| 算力 | 1280 TOPS（单芯） |
| 制程 | 5nm |
| 利用率 | 82% |
| 定位 | 车企自研芯片旗舰 |

## 行业背景

理想马赫M100是车企自研芯片浪潮的代表产品之一。2026年车企自研芯片进入量产期：
- [[byd-xuanji-a3]] - 比亚迪璇玑A3，4nm，700 TOPS
- [[nio]] - 蔚来神玑NX9031，搭载乐道L60中高配
- [[xiaopeng]] - 小鹏图灵芯片
- [[li-auto]] - 理想马赫M100

## 竞争格局

车企自研芯片对应冲智驾供应商格局：
- [[nvidia]] Orin X 在华份额从39%降至25%
- [[horizon]] 地平线基础ADAS份额达47.7%，征程6P单芯560 TOPS

## 技术意义

1. **算力突破**：单芯1280 TOPS超越双Orin X方案
2. **利用率领先**：82%利用率远超行业平均60-70%
3. **成本可控**：自研降低对外依赖，提升供应链议价权
