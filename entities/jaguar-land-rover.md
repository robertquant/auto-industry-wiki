## 概述

捷豹路虎是英国豪华汽车品牌，2008年被印度塔塔汽车收购。2026年在中国市场遭遇销量崩塌，电动化转型进展缓慢。

## 销量崩塌

### 2025年中国市场
- 捷豹在华销量14217辆（-17.9%）
- 路虎在华销量12302辆（-39.2%）
- **合计不足3万辆**，巅峰期曾逼近20万辆
- 路虎揽胜极光L官降最高44%（19万元）

### 市场地位
- 中国市场份额几乎可忽略
- 品牌影响力急剧下滑
- 经销商网络萎缩

## 电动化转型

### "重塑未来"战略
- 未来五年投资150亿英镑
- 2025年9月：国产捷豹XFL/XEL/E-PACE全线停产
- 捷豹品牌将成为纯电动豪华品牌

### 产品规划
- **首款纯电揽胜**：延期至2026年交付（原定2025年）
- **捷豹首款纯电GT**：续航700km
- 品牌拆分：揽胜、卫士、发现、捷豹四大独立品牌家族

### 与奇瑞合作
- 推出"神行者"电动车品牌
- 奇瑞主导，自建渠道
- 借中国技术造电动车

## 中国战略调整

### 人事变动
- 潘庆升任全球采购董事
- 韩少帅接任中国区CEO

### 双引擎驱动模式
- 战略规划 + 日常运营分离
- 寻求中国资本合作

## 核心问题

| 问题 | 说明 |
|------|------|
| 销量崩塌 | 中国销量从20万跌到3万辆以下，Q3仅1.7万辆（-18.4%） |
| 品牌老化 | 年轻消费者认知度低 |
| 电动化落后 | 远落后于BBA和中国品牌，纯电揽胜延期 |
| 定位尴尬 | 价格下探但品牌力不足 |

## 待观察

- 神行者品牌能否挽救中国市场
- 纯电揽胜的市场表现
- 是否会引入中国资本入股

### 2026年5月中旬动态

- **纯电揽胜/捷豹再次推迟至2026年**：电动化转型进展缓慢
- **Freelander首款SUV路测**：基于奇瑞E0X平台，中国合作模式落地
- **判断**：捷豹路虎电动化严重滞后，神行者品牌是中国技术输出的典型案例，但能否挽救中国市场仍存疑

### 2026年：电动化时间表与中国合作

- **2030年全面电动化**：捷豹路虎宣布2030年全电动化目标
- **Freelander（神行者）电动车**：与奇瑞合作，专为中国市场开发
- **2025年网络攻击**：停产**5周**，严重影响生产交付
- **揽胜150万+细分市场**：连续**3年**销冠
- **欧盟警告**：警告欧盟"欧洲制造"规则或损害供应链
- **判断**：捷豹路虎电动化时间表激进但执行力存疑；奇瑞合作是"借中国技术自救"的模式验证

### 2026年6月下旬：纯电路虎首发确认

- **Type 01纯电GT 10月首发**：纯电揽胜首款车型
- **Freelander（神行者）奇瑞合作**：专为中国市场的增程/纯电车型
- **判断**：
  - Type 01是捷豹路虎电动化转型的关键产品
  - 奇瑞合作神行者验证中国技术输出欧洲豪华品牌模式
  - 2030年全面电动化时间表依然激进
