# 轻舟智航 (Qingzhou)

中国智驾方案商，专注高阶辅助驾驶（NOA）市场。

## 2026年市场地位

### 市场份额
- **NOA市场份额：30.1%**（2026年Q1）
- **累计搭载量：突破100万台**（2026年6月）
- 中国智驾方案商第一梯队成员

### 竞争格局
| 方案商 | 市场份额 | 特点 |
|--------|----------|------|
| Momenta | 60.2% | 市占率第一，宝马/奔驰合作 |
| 轻舟智航 | 30.1% | NOA领域第二 |
| 华为ADS | - | 智选车专属，L3领先 |
| 小鹏自研 | - | 端到端激进路线 |

## 技术路线

轻舟智航属于高性价比智驾方案路线，与Momenta形成差异化：
- 专注NOA（导航辅助驾驶）场景
- 性价比定位，覆盖中端车型市场
- 与主机厂深度合作，提供全栈解决方案

## 行业定位

- 中国智驾方案商"一超多强"格局中的"强"者
- 与[[momenta]]形成市场双雄（合计90%+份额）
- 区别于[[huawei-ads]]的智选车专属模式
- 区别于车企自研（如[[xiaopeng]]、[[xiaomi]]）
