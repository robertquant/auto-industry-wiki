# 日产 (Nissan)

## 概述
日本汽车制造商，雷诺-日产-三菱联盟核心成员。2024-2026年陷入严重财务危机，经历两年巨额亏损后启动战略收缩。

## 关键事实

### 财务危机（2024-2026）
- **两年净亏损**：约1.2万亿日元（约500亿人民币）
- **生存危机**：2026年初被行业分析机构评为"最可能破产的主流车企"
- 月度现金流紧张，被迫出售资产维持运营

### 战略收缩（2026年）
- **产品线精简**：从现有车型缩减至45款
- **发布长期愿景**：聚焦核心市场、核心车型
- **中国Q1同比+7.2%**：唯一增长市场，但体量不足以扭转整体颓势

### 本田合并失败（2025年初）

- 本田-日产合并谈判
掣
- 日产寻求独立生存路径
- 三菱推进与日产、本田三方美国联合生产方案

### 日产-本田合并磋商（2026年5月）

- **合并目标**：2026年8月达成合并协议
- **三菱可能性**：可能加入形成三方联盟
- **背景**：日产财务困境持续，独立生存难度大
- **战略意义**：若合并成功，将形成全球第三大车企

## 联盟关系演变

### GT-R停产（2026年6月）

- **标志性事件**：GT-R停产，标志日产彻底放弃内燃机投资
- **历史意义**：GT-R是日产技术象征"战神"，停产意味着一个时代的结束
- **战略信号**：日产明确将全部资源投入电动化转型
- **判断**：传统性能车时代的终结，日产选择"全赌电动车"而非像丰田的混动+电动双轨策略

### 欧洲战略撤退（2026年6月）
- **CEO宣布**：不再为欧洲市场研发专属车型
- **桑德兰工厂**：一条生产线被奇瑞接手，2027年4月投产
- **战略转向**：从欧洲本土化生产转向全球车型输入
- **判断**：日产在欧洲的收缩是中国品牌出海的历史性机会窗口
- **相关实体**：[[chery]] - 接手日产全球工厂

## 联盟关系演变

### 雷诺-日产联盟
| 时间 | 事件 |
|------|------|
| 1999 | 雷诺收购日产36.8%股份，组建联盟 |
| 2023 | 雷诺将28.4%日产股份转入法国信托 |
| 2026.5 | 双方持股统一调整为各持15% |

### 联盟现状
- **联盟本质判断**："体面分手"，双方都在为后联盟时代做准备
- 雷诺从日产抽身聚焦Ampere（后转为技术中心）
- 日产押注雷诺电驱平台但避免深度绑定
- 三菱推进与日产、本田三方美国联合生产方案

### 技术依赖
- 日产计划向雷诺Ampere投资最多15%（后随Ampere撤销取消）
- 借力雷诺电驱平台降低开发成本
- 上海ACDC研发中心负责日产A级EV开发

## 2026年三座大山危机

### 美国关税冲击
| 指标 | 数据 |

### 中东地缘冲击
- 中东占日产出口约10%+
- **1400辆途乐被迫转销美国** → 加剧北美市场内卷
- 高毛利车型无法交付，利润承压

### 中国市场
- **2026年Q1同比+7.2%**：唯一增长市场
- 但体量不足以扭转整体颓势

### 风险评级
🔴 **高危** — 连续两年亏损，关税几乎抹平盈利，生存危机

## 风险评估

### 生存挑战
- 连续两年亏损后现金储备严重消耗
- 关税吞噬几乎全部盈利
- 车型精简至45款意味着退出部分细分市场
- 中国市场是其唯一增长亮点，但体量不足
- 雷诺战略转向后，技术获客渠道收窄
- 中东冲突导致高毛利车型滞销

### 战略困境
- 独立发展需要巨额研发投入，财务状况不支持
- 寻求外部合作选项有限
- 电动化转型滞后
