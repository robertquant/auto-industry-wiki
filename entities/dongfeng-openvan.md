# 东风OpenVAN

东风汽车L4无人物流车产品线，发布四款L4无人物流车型。

## 产品定位

- L4级无人物流配送车辆
- 四款车型覆盖不同场景
- 订单超4000台（截至2026年）

## 市场表现

- 2025年无人物流车行业保有量3.7万台
- 2026年Q1出货近万台
- 东风OpenVAN订单超4000台占比约10%

## 竞争格局

与[[jiushi-intelligence]]不同，东风OpenVAN是车企主导的车辆+系统整体方案：

| 模式 | 代表 |
|------|------|
| 车辆整体方案 | 东风OpenVAN |
| Inside系统供应 | 九识智能 |
| 运营服务商 | 顺丰、京东等 |

## 行业背景

无人物流车成为L4级自动驾驶商业化落地最快场景：
- 低速场景风险可控
- 固定路线/园区环境
- 快递/商超需求刚性
