# 宝马 (BMW)

## 概述
德国豪华汽车制造商,转型节奏最稳健的欧洲车企,采用双平台并行策略平滑过渡。

## 关键事实

### Neue Klasse新世代(2026年落地)
- **iX3长轴距版**:2026年沈阳量产,集成第六代电驱、大圆柱电池、全景iDrive
- **i3长轴距版**:2026年北京车展全球首发
- 800V高压架构+**108.7kWh第六代大圆柱电池**
- **CLTC续航920公里、400kW超充、Momenta智驾方案**
- 2026年Q4中国量产上市
- **下一代X5将提供五种动力总成**(含2028年氢燃料电池)

### 双平台并行策略
- **CLAR平台**:现有车型继续产出,服务当前市场
- **Neue Klasse (NCAR)**:新世代架构,2026年起逐步替代
- **2026年推约20款新车**,中国团队主导数字体验研发

### 中国本土化合作(最广泛)
| 合作方 | 领域 |
|--------|------|
| Momenta | 面向中国的高阶智驾,2026年首搭新世代车型 |
| 华为 | "中国AI"生态 |
| 阿里巴巴 | 智能座舱语音 |
| DeepSeek | 大模型 |

### 市场表现
- **2025年在华销量降12.5%（62.56万辆），连续两年两位数跌幅**
- 电气化车型47万辆（+15%），BEV占32.3万辆
- 电气化车型占总销量超25%
- Q3汽车部门利润率5.2%（vs 2024年同期2.3%）
- **2025年全球纯电44.2万辆，占比17.9%，连续两年豪华品牌电动第一**
- **BBA电动化排序：宝马 > 奔驰 > 大众**

### 技术亮点
- **"驾控超级大脑"(Heart of Joy)电子架构**
- 新世代iX3被称为"最懂中国"的宝马车型

### BMW驾控超级大脑（2026年6月）
- **BMW M新世代概念车搭载**
- 四电机独立驱动/制动精准调控
- 车辆动态控制毫秒级响应
- 驾控与智能化的深度融合

## 战略判断
宝马是欧洲五大里转型节奏最稳的——不激进押注BEV，不放弃ICE现金牛，双平台策略平滑过渡。中国市场策略务实：直接绑定中国生态，不硬推自己不成熟的软件。

### BBA电动化领先原因
- **本土化最深**：Momenta+鸿蒙+阿里+DeepSeek四重合作
- **Neue Klasse落地节奏最清晰**：2026沈阳投产，中国首发
- **电动化销量数据最扎实**：连续两年豪华品牌电动第一

- Neue Klasse能否在欧洲BEV市场建立优势
- 中国本土化合作的量产效果
- 与奔驰/大众电动化竞争格局

### 2026年5月下旬：Neue Klasse正式落地

- **Neue Klasse平台量产**：2026年在中国沈阳量产落地
- **iX3长轴距版首发**：中国专属开发，Momenta智驾方案
- **三方合作模式确认**：Momenta智驾 + 阿里AI + 鸿蒙OS，欧洲豪华品牌本土化最深
- **判断**：宝马是BBA中唯一实现智驾、AI、生态三层本土化的品牌

### 2026年6月：新世代中国首发

- **iX3长轴距版**：2026年沈阳投产，获**德国金方向盘奖**
- **Momenta智驾**：合作方案2026年量产，**首发国产iX3**
- **德国之外最大研发网络**：四大基地（北京、上海、沈阳、南京）+ 三家软件公司
- **判断**：宝马是德系三强中智驾本土化最坚决的一家，选择Momenta而非华为/自研，路线清晰

### 2026年6月下旬：全球首发与高性能路线

- **Neue Klasse iX3韩国首发**：全球亮相活动选址韩国
- **800V架构确认**：全面搭载高压快充平台
- **四电机纯电M3预演**：基于"驾控超级大脑"架构，四电机独立驱动
- **判断**：
  - 800V是德系豪华品牌电动化标配，400V时代终结信号明确
  - 四电机M3标志宝马将电动化延伸至高性能领域
  - 韩国首发体现亚太市场战略权重提升

### 2026年7月：Neue Klasse切换态度坚决

- **停产计划**：i3、i5、iX1将于2026年7月停产，为Neue Klasse让路
- **iX3长轴距版首发**：搭载Momenta端到端智驾，805km续航，400kW快充
- **战略意义**：
  - 首次在华采用本土供应商智驾方案，是德系豪华品牌的破冰之作
  - 停产现有电动车型显示出对Neue Klasse的绝对信心
  - 与大众智驾全面切换小鹏形成对标，宝马选择Momenta
- **判断**：宝马Neue Klasse切换比奔驰更坚决，本土化合作更深入
