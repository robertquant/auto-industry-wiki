# 易捷特（eGT）

## 概述
易捷特新能源汽车有限公司是由东风、雷诺、日产三方合资成立的电动车生产企业，是中国技术输出欧洲的重要案例。

## 基本信息

| 维度 | 数据 |
|------|------|
| 股权结构 | 东风50% + 雷诺25% + 日产25% |
| 生产基地 | 湖北十堰 |
| 成立时间 | 2017年 |
| 核心产品 | Dacia Spring（出口欧洲）、东风雷诺e诺（国内） |

## 业务模式

### 主营产品
- **Dacia Spring**：欧洲市场畅销平价EV，国内售价6.18-7.18万，欧洲售价15.57万起
- 累计出口欧洲超10.8万辆

### 易捷特模式特点
1. **中国生产**：利用中国供应链成本优势
2. **外资品牌**：使用雷诺集团旗下Dacia品牌
3. **全球销售**：主攻欧洲市场，规避欧盟关税壁垒

## 战略意义

### 对雷诺
- 获得<€20k平价EV产品线（Twingo项目借鉴此模式）
- 无需在欧洲建设低价EV产能
- 供应链成本优势明显（中国电池+零部件）

### 对东风
- 利用闲置产能（原东风雷诺工厂）
- 合资分红收益
- 积累电动车出口经验

### 模式复制
- **神龙汽车**（东风×Stellantis）：2026年5月获80亿注资，将投产标致/Jeep新能源，全球销售
- 本质相同：中国生产+外资品牌+全球销售

## 行业影响

易捷特模式成为中国汽车技术输出欧洲的新范式：
- 规避欧盟反补贴关税（品牌归属外资）
- 利用中国成本优势
- 适应欧洲平价EV需求（€20-25k价位段）
