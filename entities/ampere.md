# Ampere

## 概述
雷诺集团原独立电动车部门，2024年计划分拆IPO，估值曾达100亿欧元。2026年7月正式撤销独立部门定位，转型为电动与软件技术工程中心。

## 历史脉络

| 时间节点 | 事件 |
|----------|------|
| 2021 | Luca de Meo上任后推动部门独立 |
| 2023 | Ampere品牌正式发布，定位纯电业务单元 |
| 2024初 | IPO计划启动，估值目标100亿欧元 |
| 2024末 | IPO计划搁置，市场估值环境恶化 |
| 2026.7 | 正式撤销独立部门，回归雷诺工程体系 |

## 战略调整分析

### 撤销原因
- **估值环境不支持IPO**：2024年后电动车估值回调，100亿欧元目标难以实现
- **分拆模式失宠**：大众、奔驰等车企纷纷回撤独立电动车计划
- **协同效率考量**：电动与燃油车共享平台更能降本增效
- **新CEO务实路线**：François Provost推翻de Meo的分拆战略
- **EV需求低于预期**：2026年欧洲电动车市场增速放缓

### 调整后定位
- 转型为软件+EV工程技术中心
- 不再独立运营，回归雷诺集团技术体系
- 继续承担电动车型研发职能
- **11,000员工编制不变，无裁员计划**

## 战略启示
Ampere的撤销标志着欧洲车企"电动车特区"模式的终结。2021-2023年的 splitter 浪潮（大众Trinity、奔驰Ampere、雷诺Ampere）在估值回调后全部回撤。电动车不再需要独立组织，整合回主线反而能更好协同。
