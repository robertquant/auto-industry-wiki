# 地平线 (Horizon)

中国智驾芯片领军企业，提出智驾技术"三阶段"演进路线。

## 技术路线：三阶段方案

地平线提出的智驾技术演进框架：
1. **感知驱动阶段**: 传统规则+传感器融合
2. **数据驱动阶段**: 大模型、数据闭环为核心
3. **端到端阶段**: 神经网络统一感知、决策、规划

## 竞争格局

- 与 [[xiaopeng]] X-Cache世界模型方案、[[im-ad]] 端到端路线形成差异化
- 作为芯片供应商支撑下游车企智驾方案落地
- 与 [[blacksesame]] 同为国产智驾芯片双雄

## 2026年危机：市值暴跌与客户变对手

### 市值暴跌
- **股价**：从峰值11.32港元跌至5.29港元（2026年6月）
- **市值蒸发**：约880亿港元
- **核心冲击**：比亚迪璇玑A3发布，客户变对手

### 客户变对手逻辑
- 比亚迪是征程6首发客户（21款车型计划）
- 比亚迪自研璇玑A3（700 TOPS），3颗并联2100 TOPS
- 车企自研趋势对第三方芯片供应商构成结构性威胁
- 星空6P（5nm/650TOPS）尚未量产，对手已翻牌

### 征程5 CPU短板
- **问题**：J5用8核A55小核（30K DMIPS），规划控制跑不动
- **升级**：征程6升级到A78AE大核（350K+ DMIPS），提升10倍+
- **诊断**：J5不是失败产品，是有明确短板的过渡产品
- **教训**：感知需要NPU，规划控制需要CPU，CPU短板成为系统瓶颈

## 2026年进展：国产化加速


- **征程6系列量产破1000万套**：覆盖400+车型
- **征程6P**：560 TOPS，成为国产最大算力智驾芯片
- **高阶NOA芯片份额**：英伟达Orin 48%、华为MDC 660 24%、地平线征程6 17.88%
- 外资车企开始采用中国芯片方案（大众×地平线合作）
- 与英伟达形成中美智驾芯片竞争格局

### 2026年6月：代际颠覆与舱驾融合

**征程7P (J7P)：超越英伟达Thor-X**
- **最高性能版目标算力大幅超越英伟达Thor-X**
- 标志中国智驾芯片从"追赶英伟达"转向"超越英伟达"
- 国产芯片进入代际竞争阶段

**星空6P：舱驾融合突破**
- **中国首款舱驾融合智能体芯片**
- **650 TOPS BPU算力**
- **2026年Q2量产**
- 面向10-20万元车型市场
- 一颗芯片同时支撑座舱和智驾

**技术路线：E2E+VLA+世界模型**
- 地平线提出端到端+VLA+世界模型融合范式
- 与千里科技等合作伙伴共同推进
- 详见 [[driving-tech-fusion-paradigm]]
