# 华阳集团（Foryou）

中国领先的汽车电子 Tier 1 供应商，总部位于广东惠州。在智能座舱领域提供多模态融合方案，与高通、联发科等芯片厂商深度合作。

## 基本信息

| 属性 | 值 |
|------|-----|
| 全称 | 惠州市华阳集团股份有限公司 |
| 总部 | 广东惠州 |
| 成立 | 1993年 |
| 上市 | 深交所（002906） |
| 主营 | 汽车电子、精密压铸 |

## 业务板块

| 板块 | 内容 |
|------|------|
| 智能座舱 | 座舱域控制器、中控屏、HUD |
| 智能驾驶 | ADAS、环视系统 |
| 精密压铸 | 汽车精密零部件 |

## 技术路线

### 多模态融合方案
- **语音+手势+触觉+视线**融合交互
- 差异化：深度绑定整车厂，提供定制化方案
- 芯片平台：高通、联发科、瑞萨

### 产品矩阵
- 座舱域控制器
- AR-HUD（增强现实抬头显示）
- 数字仪表
- 流媒体后视镜
