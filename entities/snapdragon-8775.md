# 高通骁龙8775 (Snapdragon SA8775P)

## 概述
高通骁龙8775P（SA8775P）是高通推出的舱驾融合芯片，单芯片同时承载座舱（IVI/仪表/HUD）与智能驾驶（感知/规划/控制）的计算任务，是全球首款量产的舱驾融合方案。

## 核心规格

| 参数 | 数值 |
|------|------|
| 算力 | 144 TOPS |
| 制程 | 5nm |
| 架构 | 八核CPU + Adreno GPU + Hexagon NPU |
| 定位 | 15-25万价格带 |
| 量产时间 | 2025Q3首发，2026年规模化 |

## 量产进展

### 2026年6月量产拐点
- **获9款车型定点**：从小批量量产进入规模化上车阶段
- **首发车型**：极狐T5（2025年Q3）
- **意义**：舱驾融合从"概念验证"进入"量产元年"

### 定点车型（部分）
- 极狐T5（首发）
- 零跑D19（双芯方案）
- 车联天下AL-A2
- 其他待公布

## 技术优势
- **软件生态成熟**：继承高通座舱芯片的Android生态
- **工具链完善**：Snapdragon Ride平台工具链
- **量产验证**：已有实车落地经验
- **成本优势**：单芯片替代双芯片，预计降本1500-4000元/车

## 竞争格局

| 竞争对手 | 芯片 | 状态 | 对比 |
|---------|------|------|------|
| 地平线 | 星空6P | 2026Q3量产 | 算力更高(650TOPS)，但量产晚一年 |
| 英伟达 | Thor | 2026+量产 | 算力最高(1000TOPS)，但成本高、功耗高 |
| 黑芝麻 | C1296 | 待定 | 规格、量产时间均未公布 |

## 战略意义
- **高通座舱优势向智驾延伸**：从智能座舱霸主扩展到智驾领域
- **传统Tier1压力加大**：分布式ECU供应商面临转型压力
- **行业架构演进加速**：从分布式ECU向中央计算+区域控制器

## 风险与挑战
- 高通非All-in汽车，战略重心在手机+AI ASIC
- 迭代速度可能被专注汽车的中国厂商追上
- 地平线星空6P 2026Q3量产将形成直接竞争
