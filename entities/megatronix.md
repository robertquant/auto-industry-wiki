# 镁佳科技（Megatronix）

## 概述
镁佳（北京）科技有限公司，中国智能座舱解决方案供应商，主要客户为奇瑞、长安等自主品牌车企。

## 基本信息

| 维度 | 数据 |
|------|------|
| 公司全称 | 镁佳（北京）科技有限公司 |
| 英文名 | Megatronix |
| 市场地位 | 智能座舱域控方案商中国第二（市占率9.3%） |
| 创始人 | 庄莉（清华CS本硕第一+UC Berkeley博士，前蔚来软件VP） |
| 2025年营收 | 14.92亿元（+5%） |
| 2025年经调整亏损 | 1.2亿元（同比收窄32%） |
| 2024年交付量 | 60.99万套域控解决方案 |
| 核心业务 | 智能座舱域控解决方案 |

### 财务转折（2024-2025）
- **主动砍掉德赛西威**：德赛西威曾占53%营收，2024年被砍掉
- **转型方向**：从Tier 2转Tier 1，直接面对主机厂
- **毛利率变化**：12.1% → 24.5%，翻倍
- **收入结构问题**：99%收入来自硬件销售，软件收入几乎为零

### 芯片平台策略
- **高通全家桶**：8155 → 8255 → 8295
- **8255 vs 8295差异**：
  - 8255：有ASIL安全岛+现代架构，适合智驾融合
  - 8295：极致GPU渲染，适合座舱娱乐
  - **判断**：两者不是简单升级关系，是不同定位的产品线



## 客户结构

| 客户 | 特点 |
|------|------|

**核心问题**：客户集中度高，且均为"压价狠"的自主品牌，议价能力弱。

与德赛西威、华为、博世相比，镁佳完全依赖国内自主品牌客户。

## 问题与风险

### 1. 客户结构风险
- 长安、奇瑞均以"白嫖"供应链著称
- 议价能力弱，利润率承压
- 缺乏国际化客户背书

### 2. 规模天花板
- 未进入座舱域控前十，与头部差距明显
- 德赛西威份额15.3%，镁佳难以追赶

### 3. 技术差异化不足
- 无自研芯片，依赖高通方案
- 方案同质化，难以形成护城河

## 判断

镁佳科技的问题是典型"赛道好、位置差"：智能座舱是大势所趋，但镁佳处于价值链低位，客户结构决定了利润天花板。要突破必须：
1. **国际化**：要么直接供货海外车企，要么跟随中国车企出海做本地化配套
2. **技术升级**：从方案集成走向核心软硬件能力
3. **客户多元化**：突破"白嫖厂商"死循环
