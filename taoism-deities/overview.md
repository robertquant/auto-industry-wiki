# 道教神仙体系总览

## 核心架构

道教神仙体系是一个严密的等级制度，从最高的"道"化身到民间俗神，层层分明。

**最高原则：道**
- "道"是宇宙本源
- 三清是"道"的人格化
- 一切神仙皆归于道

## 五层神仙体系

### 第一层：三清（最高神）
| 称号 | 名字 | 象征 | 居所 |
|------|------|------|------|
| 玉清 | 元始天尊 | 大道本源、创世 | 玉清境清微天 |
| 上清 | 灵宝天尊 | 大道演化、传法 | 上清境禹余天 |
| 太清 | 道德天尊（太上老君） | 大道教化 | 太清境大赤天 |

### 第二层：四御（天界统治者）
| 称号 | 职能 |
|------|------|
| 玉皇大帝 | 天界最高统治者，统领三界 |
| 紫微大帝 | 统御众星，掌管人间帝王 |
| 勾陈大帝 | 统御万雷，掌管兵革战争 |
| 后土皇地祇 | 大地之母，掌管山川土地 |

### 第三层：重要神祇
- 星辰神系：斗姆元君、北斗七星、太白金星、二十八宿
- 天界要职：王母娘娘、东华帝君、文昌帝君、真武大帝、雷祖
- 冥界神系：酆都大帝、东岳大帝、十殿阎王

### 第四层：仙真（修炼得道者）
- 八仙、上古仙人、历代天师

### 第五层：民间俗神
- 财神、门神、灶神、土地、城隍、妈祖

## 仙人五等

| 等级 | 名称 | 特点 |
|------|------|------|
| 一等 | 天仙 | 飞升天界，不老不死，合道 |
| 二等 | 神仙 | 游历人间，可隐可现 |
| 三等 | 地仙 | 居名山大川，长生不老 |
| 四等 | 人仙 | 延年益寿，未解脱生死 |
| 五等 | 鬼仙 | 死后修成，依附祭祀 |

**修道目标**：人仙 → 地仙 → 神仙 → 天仙 → 合道

## 道教神仙特点

### 与西方神话对比

| 特点 | 道教 | 希腊/北欧 |
|------|------|---------|
| 来源 | 凡人可修炼成仙 | 天生神族 |
| 性格 | 清静无欲 | 七情六欲 |
| 职责 | 掌管自然、人间秩序 | 干涉人间、争斗 |
| 形象 | 多为老者、仙风道骨 | 多为俊美青年 |
| 性别 | 重男轻女 | 男女平等 |

### 成仙途径

1. **修炼**：内丹、外丹、服食
2. **功德**：行善积德
3. **尸解**：死后元神飞升
4. **点化**：神仙度化

## 核心概念

### 三界
- **天界**：神仙居所
- **人界**：凡人世界
- **地界**：冥界、地府

### 三十六天
道教将天界分为三十六层：
- 欲界六天
- 色界十八天
- 无色界四天
- 四梵天
- 三清天
- 大罗天

### 洞天福地
十大洞天、三十六小洞天、七十二福地
- 人间仙境，修道圣地
