1. **天道体系**：三清 → 四御 → 五老 → 六御
2. **星辰体系**：七星 → 九曜 → 二十八宿
3. **冥界体系**：酆都大帝 → 十殿阎王
4. **民间体系**：八仙 → 各路俗神
