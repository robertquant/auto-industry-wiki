# 道教女神详解

## 一、西王母（王母娘娘）

---

### 基本信息

| 项目 | 内容 |
|------|------|
| 全称 | 瑶池金母、西王母、王母娘娘 |
| 地位 | 女仙之首，道教最高女神之一 |
| 职能 | 掌管女仙、长生不老、赏善罚恶 |
| 居所 | 昆仑山瑶池 |
| 配偶 | 东王公（东华帝君） |

---

### 形象特征

#### 经典形象

| 部位 | 描述 |
|------|------|
| 面容 | 端庄、美丽、威严 |
| 肤色 | 白皙 |
| 须发 | 女性形象，无须 |
| 冠饰 | 凤冠或玉冠 |
| 服饰 | 华丽凤袍，绣有凤凰图案 |
| 手持 | 蟠桃或玉如意 |

#### 《山海经》形象

| 部位 | 描述 |
|------|------|
| 面容 | 人面兽身 |
| 牙齿 | 虎齿 |
| 发型 | 蓬发 |
| 配饰 | 玉胜（玉饰）|
| 动物 | 善啸，有豹尾 |

**注**：《山海经》中的西王母形象较为原始，与后世的美丽女神形象不同。

#### 道教形象

| 部位 | 描述 |
|------|------|
| 面容 | 端庄美丽，慈祥威严 |
| 冠饰 | 凤冠，有凤凰装饰 |
| 服饰 | 金色或紫色凤袍 |
| 手持 | 蟠桃、玉如意或玉圭 |
| 坐骑 | 凤凰或鸾鸟 |

---

### 法力神通

| 神通 | 描述 |
|------|------|
| 长生不老 | 掌管长生不老之术，赐予蟠桃 |
| 赏善罚恶 | 考察人间善恶，赏善罚恶 |
| 统领女仙 | 所有女仙之首 |
| 创造万物 | 参与天地造化 |
| 预知未来 | 通晓天地万物 |
