# 新势力格局分析（审慎判断）

## 一、销量≠利润

### 销量排名

| 品牌 | 2025销量 | 同比 |
|------|---------|------|
| 零跑 | 59.7万 | +103% |
| 小鹏 | ~42.9万 | +100% |
| 理想 | 40.6万 | -19% |
| 蔚来 | 32.6万 | +47% |

### 利润对比

| 品牌 | 利润状况 | 单车净利润 |
|------|---------|-----------|
| 零跑 | 薄利多销 | **约905元** |
| 小鹏 | 2025年首次全年盈利 | - |
| 理想 | 利润率最高 | 较高 |
| 蔚来 | Q4首次季度盈利 | - |

**核心问题**：零跑销量第一，但单车利润只有905元，靠规模撑着。

## 四、审慎判断

| 观点 | 说明 |
|------|------|
| 销量第一≠格局已定 | 利润是关键指标 |
| 零跑销量最高但利润极低 | 出海Stellantis分走利润 |
| 理想利润最高但销量下滑 | 纯电转型是生死战 |
| 小鹏刚盈利 | 能否持续待观察 |
| 新势力格局远未定型 | 2026年是真正分水岭 |

## 五、不应对零跑过于乐观

| 风险点 | 分析 |
|------|------|
| 单车利润极低 | 905元，规模效应脆弱 |
| 出海分利 | Stellantis占51%，利润被分走 |
| 品牌天花板| 主品牌认知固化在性价比 |
| 第二品牌不确定 | 30万+市场竞争激烈 |
| 技术护城河不足 | 智驾不是第一梯队 |

---

## 六、2026年关键观察点

| 时间节点 | 观察点 |
|---------|--------|
| 2026Q2 | 零跑第二品牌发布 |
| 2026Q3 | 理想纯电i8/i6市场表现 |
| 2026Q4 | 零跑Stellantis渠道放量情况、利润改善情况 |
| 2027年 | 新势力格局定型 |

---

**核心结论**：新势力格局远未定型，销量只是一种指标，利润、现金流、可持续性同样重要。零跑的问题是"卖得多但赚得少"，这在价格战下是非常脆弱的。不应对任何一家过于乐观或悲观，2026年是真正的分水岭。
